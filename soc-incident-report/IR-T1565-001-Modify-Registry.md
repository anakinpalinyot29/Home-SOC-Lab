### Technique: Stored Data Manipulation â€“ Modify Registry (T1565.001)

Attackers may **modify registry values** to:
- Alter system behavior
- Disable security features
- Achieve persistence
- Manipulate application or OS configurations

Registry modifications are particularly dangerous because they:
- Often persist across reboots
- May not generate visible user-facing symptoms
- Can silently weaken endpoint security
